변경 이력

버전 1.4.0 (2024-10-29)

  • 키보드 0~9까지 입력 후 클릭하면 해당 숫자부터 시작하는 기능이 추가되었습니다
  • Shift키를 눌러 입력한 숫자를 뒤로가기 후 입력 시 숫자가 잘 못 입력되는 Bug를 수정했습니다

버전 1.3.0 (2024-10-17)

  • 숫자입력 모드에서 Shift키를 누른채 클릭 시 동일한 숫자가 입력되는 기능이 추가되었습니다

버전 1.2.0 (2024-10-15)

  • 다국어 지원(한국어, 영어, 일본어): 브라우저 언어에 따라서 자동으로 언어가 선택됩니다
  • 사용자 마지막 설정값 동일 브라우저로 재접속 시 유지 기능이 추가되었습니다

버전 1.1.0 (2024-09-23)

  • 숫자입력 모드에서 숫자 크기를 수정할 수 있는 기능이 추가되었습니다
  • 그려지는 숫자와 도형의 색상을 지정할 수 있는 기능이 추가되었습니다
  • 되돌리기 버튼으로 이전 액션을 취소할 수 있습니다
  • 되돌리기 버튼을 단축키(Ctrl + Z, Command + Z)로 사용할 수 있습니다

버전 1.0.0 (2024-09-23)

  • 최초 출시
  • 로컬 이미지 불러오기
  • 클립보드 이미지 가져오기
  • 클릭 시 이미지 위에 숫자 그리기
  • 이미지 저장 기능
