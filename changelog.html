변경 이력

버전 1.12.2 (2025-07-09)

  • 도형 모드에서 사각형 내부에 클릭하여 숫자나 텍스트를 입력할 수 없는 문제를 해결했습니다.

버전 1.12.1 (2025-07-08)

  • 캔버스에서 우클릭 시 클릭 이벤트가 발생하는 버그를 수정했습니다.

버전 1.12.0 (2025-07-08)

  • 기본 이미지를 "how to use" 가이드로 대체하고, 캔버스에 이미지가 없을 때 안내 메시지를 표시하도록 개선했습니다.
  • 모바일 환경에서 캔버스 클릭 시 숫자가 표시되지 않던 문제를 수정했습니다. (터치 이벤트 처리 및 redrawCanvas 로직 개선)

버전 1.11.0 (2025-06-28)

  • 입력 한 숫자, 도형, 텍스트에 마우스를 올려뒀을때 손모양으로 커서가 변경되면 끌어서 위치를 옮길 수 있습니다.

버전 1.10.0 (2025-05-30)

  • 가져오는 이미지의 배경색이 흰색인 경우 이미지 경계를 확인하기 어려운 문제가 있었습니다. 이미지를 가져와서 보여주는 영역의 색상을 회색으로 변경하였습니다.

버전 1.9.0 (2025-03-27)

  • 텍스트 모드가 추가되었습니다. 희망하는 위치를 클릭하여 텍스트를 입력할 수 있습니다.
  • 디자인이 개선되었습니다.
  • 방문자 수 표시 기능이 제거되었습니다.

버전 1.8.0 (2025-02-15)

  • 숫자모드에서 V키를 누른채 클릭하게되면 마지막 클릭한 기준으로 가로 위치 값이 같고 세로 값의 위치만 다르게 클릭됩니다.
  • 숫자모드에서 H키를 누른채 클릭하게되면 마지막 클릭한 기준으로 세로 위치 값이 같고 가로 값의 위치만 다르게 클릭됩니다.

버전 1.7.0 (2025-01-17)

  • 도형모드에서 '화살표'를 이용할 수 있습니다. 가져온 이미지에 마우스로 드래그하여 화살표를 넣을 수 있습니다.

버전 1.6.0 (2024-12-13)

  • 이미지 크기 옵션이 추가되었습니다. 숫자, 도형 작업 후 이미지 크기 조정을 할 경우 작업 내용이 크거나 줄어들지 않고 초기화 됩니다.

버전 1.5.0 (2024-12-11)

  • 높이가 900px 이상인 이미지를 가져왔을 때 자동으로 900px로 리사이즈 처리되는 기능을 추가했습니다.

버전 1.4.1 (2024-11-01)

  • Annotate 작업 후 새로운 이미지를 가져올 때 숫자가 초기화되지 않고 이어지는 Bug를 수정했습니다

버전 1.4.0 (2024-10-29)

  • 키보드 0~9까지 입력 후 클릭하면 해당 숫자부터 시작하는 기능이 추가되었습니다
  • Shift키를 눌러 입력한 숫자를 뒤로가기 후 입력 시 숫자가 잘 못 입력되는 Bug를 수정했습니다

버전 1.3.0 (2024-10-17)

  • 숫자입력 모드에서 Shift키를 누른채 클릭 시 동일한 숫자가 입력되는 기능이 추가되었습니다

버전 1.2.0 (2024-10-15)

  • 다국어 지원(한국어, 영어, 일본어): 브라우저 언어에 따라서 자동으로 언어가 선택됩니다
  • 사용자 마지막 설정값 동일 브라우저로 재접속 시 유지 기능이 추가되었습니다

버전 1.1.0 (2024-09-23)

  • 숫자입력 모드에서 숫자 크기를 수정할 수 있는 기능이 추가되었습니다
  • 그려지는 숫자와 도형의 색상을 지정할 수 있는 기능이 추가되었습니다
  • 되돌리기 버튼으로 이전 액션을 취소할 수 있습니다
  • 되돌리기 버튼을 단축키(Ctrl + Z, Command + Z)로 사용할 수 있습니다

버전 1.0.0 (2024-09-23)

  • 최초 출시
  • 로컬 이미지 불러오기
  • 클립보드 이미지 가져오기
  • 클릭 시 이미지 위에 숫자 그리기
  • 이미지 저장 기능
